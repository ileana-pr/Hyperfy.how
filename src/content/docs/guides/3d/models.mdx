## <u>Generators</u>


### Meshy
Meshy is an AI 3D model generator that helps to effortlessly transform images and text into 3D models in seconds.
* Features:
  + AI 3D model generation
  + Image to 3D model
  + Text to 3D model

### Stable Point Aware 3D (SPAR3D)
<Badge text="Stability-AI" variant="tip" />

A state-of-the-art open-source model for fast feedforward 3D mesh reconstruction from a single image using point cloud conditioning.
* Features:
  + 3D mesh reconstruction
  + Single image input
  + Point cloud conditioning
  + Open-source

<LinkCard title="SPAR3D Documentation" href="https://github.com/Stability-AI/stable-point-aware-3d/blob/main/README.md#comfyui-extension" />
